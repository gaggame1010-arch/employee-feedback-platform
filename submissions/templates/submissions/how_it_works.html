How it works

A simple, anonymous workflow for employees and HR — no employee accounts required.

For employees

1 Open the platform and choose Submit.

2 Enter the company access code (shared internally).

3 Select a type: Issue, Concern, Question, or Suggestion.

4 Write your message. Avoid names or identifying details unless necessary.

5 You receive a receipt code to check updates later.

Submit anonymously Check status

For HR

1 Log in to the HR Dashboard.

2 Review and filter submissions by type/status/date.

3 Respond from the dashboard. The employee can see your reply using their receipt code.

4 Update status: New → In Review → Responded → Closed.

HR login

Anonymity, explained

The platform is designed so submissions are stored without employee identity. Access is controlled by (1) a shared company access code for submissions and (2) staff-only login for HR.
